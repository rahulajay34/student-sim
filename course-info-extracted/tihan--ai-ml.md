# Module-1: Introduction to AI, Python Basics, and Machine Learning Fundamentals

- Understand AI concepts, types of learning (supervised, unsupervised, reinforcement), & core algorithms
- Learn data types, functions, loops, and essential libraries like NumPy and Pandas for data handling
- Master linear algebra, statistics, and probability tailored for machine learning applications
- Explore Scikit-learn, TensorFlow, PyTorch, and understand autonomous drone systems

# Module-2: Data Handling, Visualization, and ML Techniques

- Master data collection, cleaning, preprocessing, and handling missing values, outliers, and duplicates
- Visualize data with Matplotlib and Seaborn, uncovering insights and patterns
- Learn techniques for feature extraction, scaling, encoding, and transformation for ML models
- Explore key regression and classification algorithms and understand sensor roles in Unmanned Aerial Vehicles (UAVs)

# Module-3:Predictive Modeling, Regression, and Evaluation

- Implement regression models for continuous variable prediction
- Master linear, multiple, and polynomial regression techniques
- Evaluate model performance using MAE, MSE, and R² metrics
- Apply cross-validation to ensure model robustness and generalizability

# Module-4: Classification Algorithms and Deep Learning with Drone Technology Insights

- Implement classifiers like logistic regression, k-NN, SVM, and decision trees for categorizing data
- Build neural networks and deep learning models for complex pattern recognition and feature learning
- Train models using backpropagation, optimize through hyperparameter tuning, and evaluate classification accuracy
- Apply drone navigation techniques including GPS, visual odometry, and sensor fusion for position tracking

# Module-5: Model Tuning, Ethics in AI, and AI Deployment with Drone Technology Insights

- Techniques like grid and random search fine-tune hyperparameters for better performance
- Regularization (L1/L2) and cross-validation ensure model generalization
- Focuses on fairness, privacy, interpretability, and transparency in AI systems
- Drones communicate wirelessly with control stations and each other via Wi-Fi, LTE, and radio frequencies

# Module-6: Capstone Project — SmartStream Recommender

- Build a Netflix-style recommendation engine using collaborative filtering techniques
- Implement content-based and hybrid recommendation approaches for improved accuracy
- Deploy your recommender system as a web application with user feedback functionality
- Present your system's performance metrics and explain your design decisions

## FAQs

### Program Overview

**Q: What is the program duration?**

The program spans 6 months, offering a structured and immersive learning journey designed to equip you with in-demand Artificial Intelligence, Machine Learning, and drone technology skills.

**Q: What topics and concepts are covered in the program?**

The program covers AI and Machine Learning across five modules and Capstone Project (5 weeks each), including AI fundamentals, Python programming (data types, functions, NumPy, Pandas), mathematics for ML (linear algebra, statistics), and ML libraries (Scikit-learn, TensorFlow, PyTorch). It includes data handling, visualization (Matplotlib, Seaborn), feature engineering, predictive modeling (regression, decision trees), classification algorithms (logistic regression, SVM, k-NN), deep learning (neural networks), model tuning (grid search, regularization), and ethical AI (fairness, privacy). Drone technology topics include autonomous drone systems, sensor perception, localization, and wireless communication. Hands-on projects reinforce practical skills.

**Q: What are the projects I will work on?**

The capstone project, SmartStream Recommender, involves building a Netflix-style recommendation engine using a public ratings dataset, with tasks like data cleaning, feature engineering, implementing recommendation algorithms (popularity-based, content-based, collaborative filtering), evaluating models, auditing bias, and deploying via Streamlit. Other projects include Python programming, dataset cleaning, exploratory data analysis, and building regression/classification models for drone-related tasks (e.g., flight metrics prediction, aerial image classification).

**Q: Will I receive a digital or physical copy of the programme certificate?**

Upon successful completion of the program and fulfillment of all requirements, participants will receive a digital copy of the certificate. The digital certificate will be dispatched to the registered address shared during enrollment.

**Q: When will I receive my certificate after completing the program?**

Certificates will be issued within 2–3 months from the date of the final examination. The exact timeline for issuance will be communicated during the program.

**Q: What is the mode of delivery for the program?**

The program is delivered online through the Zoom platform, featuring live sessions, practical projects, and tutorials.

**Q: Who will be teaching the classes in this program?**

The sessions will be led by esteemed IIT faculty and industry mentors who bring extensive expertise in AI, Machine Learning, and drone technology, ensuring a blend of academic rigor and practical insights tailored to real-world applications.

**Q: What tools and software will I learn during the course?**

You will gain hands-on experience with industry-standard tools such as Python, NumPy, Pandas, Scikit-learn, TensorFlow, PyTorch, Matplotlib, and Seaborn for AI-ML development and data visualization. Additional tools may include Streamlit for deploying models and software for drone data analysis (e.g., ROS for drone simulation).

**Q: Will I work on real-world case studies?**

Yes, the program includes real-world case studies that enable you to apply AI, ML, and drone technology concepts to practical scenarios, helping you build hands-on experience in solving industry-relevant challenges.

**Q: What is campus immersion?**

The 2-day campus immersion is an optional in-person immersive experience designed to enhance learning through instructor interaction, campus exposure, and networking opportunities. The valedictory ceremony will be conducted as part of the campus immersion. Details regarding the schedule, venue, and participation guidelines will be communicated to participants during the program.

**Q: Is campus immersion mandatory, and when does it take place?**

It is typically scheduled towards the end of the program or after completion. While not mandatory, participation is highly recommended. The schedule will be shared in advance to enable travel planning.

**Q: What is the fee for the campus immersion?**

The campus immersion fee is ₹5,000 per participant per day. This fee covers only the arrangements at the immersion venue, including lunch and refreshments provided during the scheduled immersion hours.<br><br>The immersion fee is not included in the programme fee and must be paid separately by participants.
