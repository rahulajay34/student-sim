Module 1: Python Programming Fundamentals and Flow Control (3 Weeks)

- • Master core Python syntax including data types, variables, and string manipulation
- • Organize information efficiently using lists, dictionaries, tuples, and sets
- • Implement complex decision-making logic using if/else statements and loops
- • Develop clean and reusable code through custom function definitions

Module 2: The Data Handling Toolkit: NumPy and Pandas (3 Weeks)

- • Leverage NumPy for high-speed numerical operations and vectorization
- • Master the Pandas DataFrame for structured data import and inspection
- • Apply professional data cleaning techniques to handle missing values and formatting
- • Perform advanced data manipulation using grouping, filtering, and merging

Module 3: Statistical Thinking, Data Preparation, and ML Workflow (4 Weeks)

- • Calculate and interpret key descriptive statistics to understand data distributions
- • Create professional visualizations using Matplotlib and Seaborn
- • Apply feature scaling and engineering techniques like normalization and standardization
- • Define sequential steps for building reliable machine learning pipelines
- • Implement the vital train-test-validation split for model reliability

Module 4: Core Supervised Learning: Regression and Classification (5 Weeks)

- • Implement and evaluate simple and multiple linear regression models
- • Master regression metrics including Mean Squared Error and R-squared
- • Build classification models using logistic regression and decision trees
- • Evaluate categories using confusion matrices, precision, recall, and F1-scores

Module 5: Advanced ML concepts and Unsupervised Learning (5 Weeks)

- • Address model robustness through the bias-variance trade-off and cross-validation
- • Implement ensemble methods like random forests and master gradient boosting intuition
- • Optimize model settings for peak performance using hyperparameter tuning
- • Discover hidden patterns and segment data using K-means clustering (unsupervised learning)

Module 6: Advanced Generative AI Concepts and Tools (4 Weeks)

- • Explore the landscape of foundation models including GPT, Gemini, and Llama
- • Master advanced prompt engineering techniques like chain-of-thought and few-shot prompting
- • Understand RAG architectures for building grounded and accurate AI applications
- • Build autonomous AI agents using planning and tool-usage frameworks
- • Learn how vector databases and embeddings enable semantic search

Module 7: Capstone Project and Final Presentation (2 Weeks)

- • Finalize project scope and perform end-to-end data sourcing and preparation
- • Select and execute appropriate machine learning models for real-world problems
- • Conduct rigorous model evaluations and document the entire analytical process

You train a small CNN on a public Indian e-commerce image dataset using free Colab GPU, push validation accuracy past 85%, and produce a confusion-matrix breakdown that names exactly which category pairs the model still fumbles — and why.

Computer vision
TensorFlow / PyTorch
CNN architecture
Colab GPU
Confusion-matrix analysis

Build a churn-prediction model for a D2C customer dataset. Engineer RFM features (recency, frequency, monetary), train a gradient-boosting classifier, deploy as a FastAPI endpoint, and write the model card — including the ethical considerations around retention targeting that any real product team will probe in your interview.

XGBoost
RFM feature engineering
FastAPI deployment
Model card
Ethical framing

DECISION OUTPUT
A deployable churn predictor with a documented model card you can defend to a hiring panel.

## FAQs

### Programme Overview

**Q: What is the duration of the program?**

The program spans 6 months, offering a comprehensive learning journey designed to equip participants with practical skills in Artificial Intelligence and Machine Learning.

**Q: Will I receive a digital or physical copy of the programme certificate?**

Upon successful completion of the program and fulfillment of all requirements, participants will receive a digital copy of the certification on their registered email ID.

**Q: When will I receive my certificate after completing the program?**

Certificates will be issued within 2–3 months from the date of the final examination. The exact timeline for issuance will be communicated during the program.

**Q: What is the mode of delivery for the program?**

The program is offered online, allowing participants to attend live sessions via the Zoom platform. This enables participation from anywhere and direct interaction with faculty and peers.

**Q: Who will be teaching the classes in this program?**

The sessions will be delivered by esteemed faculty from IIT Roorkee and other leading IITs, along with top industry experts. This combination ensures strong academic rigor while providing real-world perspectives, so you learn how Artificial Intelligence, Machine Learning, and Generative AI are applied in industry-relevant scenarios today.

**Q: What is campus immersion?**

The 2-day campus immersion is an optional in-person immersive experience designed to enhance learning through instructor interaction, campus exposure, and networking opportunities.

**Q: Is campus immersion mandatory, and when does it take place?**

It is typically scheduled towards the end of the program or after completion. While not mandatory, participation is highly recommended. The schedule will be shared in advance to enable travel planning.

**Q: What is the fee for the campus immersion?**

There is no additional campus immersion fee. Accommodation may be provided at an additional cost, subject to availability, and may range between ₹1,200 - ₹1,500 per day per participant. Travel and food expenses must be borne by students and are not included in the course fees.

### Eligibility & Admissions

**Q: What are the eligibility criteria for the program?**

The program is open to individuals who have completed at least 12th grade or higher. It is suitable for anyone passionate about building a career in Data Science and Artificial Intelligence, regardless of prior experience in the field.

**Q: Can international students apply for the program?**

Yes, international students can apply for the program, provided they meet the eligibility criteria.

**Q: What is the admission process?**

The admission process begins with signing up for the program and taking the Qualifier Test. If you clear the test, you will be invited for a one-on-one session where you will receive further details on completing your enrollment.

**Q: What skills does the Qualifier test assess?**

The Qualifier Test assesses your Quantitative Aptitude, Logical Reasoning, Data Interpretation, and Verbal Ability. These skills are essential to succeed in the program and understand core AI and ML concepts.

**Q: Can I take the Qualifier test after my college exams are over?**

Seats for the program are limited, so it is strongly recommended to take the Qualifier Test as soon as possible. This ensures that you do not miss the opportunity to start the program on time.
