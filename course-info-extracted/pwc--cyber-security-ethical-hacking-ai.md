#### Certification

Module 1: Cybersecurity Foundations & Evolving Threat Landscapes

- • History, terminology & career tracks in cybersecurity
- • Types of attacks: phishing, malware, ransomware, social engineering
- • Cyber kill chain & MITRE ATT&CK framework
- • Introduction to adversarial AI and attack simulation using LLMs
- • Hands-on Project: Build your own threat map using open-source data

Module 2: Ethical Hacking, Footprinting & Vulnerability Assessment

- • Kali Linux, Metasploit, Nmap, Burp Suite
- • Scanning networks, enumerating systems, and finding open ports
- • Vulnerability scanning tools: Nessus, OpenVAS
- • OWASP Top 10 threats
- • Using AI for automated vulnerability classification and risk scoring
- • Hands-on Project: Recon and vulnerability report on a simulated organization

Module 3: Advanced Network Security & Incident Response

- • Firewalls, IDS/IPS, VPNs, and honeypots
- • Detecting and responding to intrusions
- • Log analysis and SIEM (Security Information and Event Management)
- • Building an AI-assisted SOC (Security Operations Center) simulator
- • Hands-on Project: Configure Snort, use Splunk + AI alerts

Module 4: Secure Coding & Application Security

- • Threat modeling & secure software design
- • Code vulnerabilities (SQLi, XSS, CSRF, etc.)
- • Static & dynamic code analysis tools
- • Using AI to scan and refactor insecure code (GitHub Copilot, DeepCode)
- • Hands-on Project: Audit and secure a buggy web application

Module 5: AI-Driven Cybersecurity: Detection, Automation & Defense

- • Introduction to machine learning in cybersecurity
- • Anomaly detection with AI (unsupervised learning)
- • AI in phishing, spoofing, and spam detection
- • Generative AI for penetration testing automation
- • Agentic AI: Creating autonomous agents for continuous network monitoring
- • Hands-On Project: Train an AI model to detect unusual traffic patterns

Module 6: Cyber Law, Compliance & Digital Forensics

- • Indian Cyber Law, IT Act, GDPR, HIPAA
- • Risk frameworks: NIST, ISO 27001
- • Chain of custody, evidence handling
- • Forensic tools: Autopsy, FTK Imager, Wireshark
- • AI for evidence triage and pattern recognition in large datasets
- • Capstone Project

Ask Me Anything Session (2 Hours)

An open forum session to address participant queries and facilitate discussion on topics covered throughout the training program. This session will be led by PwC experts and is intended to reinforce learning and clarify concepts as part of the curriculum.

## FAQs

### Program Overview

**Q: What is the program duration?**

The program spans 4 months, offering a structured and immersive learning journey designed to equip you with in-demand cybersecurity and ethical hacking skills with AI.

**Q: What topics and concepts are covered in the program?**

The program offers a comprehensive overview of cybersecurity and ethical hacking, enhanced with applications of AI. It begins with foundational concepts such as the cybersecurity landscape, threat modeling, and security frameworks. Participants will explore ethical hacking techniques, including vulnerability assessment, penetration testing, and system exploitation. Throughout the program, students engage in hands-on labs and real-world case studies to apply their learning in practical scenarios.

**Q: What are the projects I will work on?**

You’ll work on real-world cybersecurity projects like threat mapping, vulnerability assessments, securing web applications, AI-driven anomaly detection, and automated penetration testing. The capstone project involves securing a simulated fintech or e-commerce platform, covering end-to-end security implementation and reporting.

**Q: Will I receive a certificate upon completion?**

Yes, upon successful completion of the course, you will receive a Certification of Completion from PwC India, a globally recognized professional services organisation.

**Q: What is the mode of delivery for the program?**

The mode of delivery of the program is online through the Zoom platform.

**Q: Who will be teaching the classes in this program?**

The sessions will be led by expert faculty from PwC India and supported by industry practitioners. You'll learn directly from cybersecurity specialists with deep experience in both corporate and academic domains.

**Q: What tools and software will I learn during the course?**

You will gain hands-on experience with leading cybersecurity and AI tools such as Kali Linux, Metasploit, Wireshark, Burp Suite, Splunk, OpenVAS, TensorFlow, Scikit-learn, ChatGPT, and Google AutoML, used across ethical hacking, threat detection, and AI-powered defense.

**Q: Will I work on real-world case studies?**

Yes, the program includes real-world case studies that enable you to apply your knowledge to practical scenarios, helping you build hands-on experience in solving industry-relevant challenges.
